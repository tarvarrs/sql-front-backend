# sql-front-backend

### О проекте

API веб-приложения для решения задач по SQL на тематику Великой Отечественной Войны. Цель проекта – повышение патриотического воспитания у студентов IT-специальностей.

### Используемый стек

* FastAPI
* SQLAlchemy
* PostgreSQL
* JWT, PyJWT
* uvicorn
